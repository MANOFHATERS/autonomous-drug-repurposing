# ─── Phase 6 V1 launch: Literature Cross-Check (RT-005 ROOT FIX) ──────────
# DOCX §8 "Layer 2: Literature Cross-Check" requires an automated PubMed
# literature search for the top 50 novel predictions. DOCX §8 "V1 Launch
# Criteria" lists "At least 5 top predictions are supported by published
# literature" as a HARD launch gate.
#
# The literature cross-check lives in rl/rl_drug_ranker.py (search_pubmed
# via Bio.Entrez). Without biopython installed, the cross-check raises
# RuntimeError at runtime, which the scientific_validation gate catches
# and DOWNGRADES to a warning (literature_check_skipped=True). This
# silently disables a V1 launch criterion — exactly the failure mode the
# integration plan calls out as RT-005.
#
# ROOT FIX: declare biopython as a HARD dependency so the literature
# cross-check is always available. The scientific_validation gate can
# then ENFORCE the "≥5 literature-supported predictions" criterion
# instead of silently skipping it.
